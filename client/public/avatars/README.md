# Шаблонные аватарки

Здесь должны быть размещены 8 изображений аватарок для использования в форме редактирования профиля сотрудника.

## Требования к файлам:

- **Формат**: PNG или JPG
- **Размер**: Рекомендуется квадратные изображения (например, 200x200px или больше)
- **Имена файлов**: 
  - `1.png`
  - `2.png`
  - `3.png`
  - `4.png`
  - `5.png`
  - `6.png`
  - `7.png`
  - `8.png`

## Как добавить аватарки:

1. Подготовьте 8 изображений
2. Переименуйте их согласно списку выше (просто `1.png`, `2.png` и т.д.)
3. Поместите файлы в эту папку (`client/public/avatars/`)
4. Перезапустите dev-сервер (если он запущен)

Изображения будут автоматически доступны по путям `/avatars/1.png`, `/avatars/2.png` и т.д.
